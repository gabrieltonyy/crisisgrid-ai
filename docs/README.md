# CrisisGrid AI Documentation

This folder contains project documentation that was previously spread across the repository root, the legacy prompt-context folder, `Buildproject/`, and frontend implementation-summary files.

## Structure

- `context/` - architecture, product, schema, security, testing, agent, prompt-engineering, and hackathon context documents.
- `planning/` - build plans, task lists, and implementation plans.
- `ibm/` - IBM product access and integration requirement documents.
- `prompts/` - master prompt and AI-assisted build instructions.
- `frontend/implementation/` - frontend module implementation summaries.
